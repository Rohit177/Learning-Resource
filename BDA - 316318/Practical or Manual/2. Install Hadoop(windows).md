### Install Hadoop on Windows

* **From resource attached link:**  https://drive.google.com/drive/folders/1i9lkZcs9t9GLWMgraNLb3T4j_Rx1xxU_?usp=sharing

### 1. Prerequisites
* **Java JDK 8:** (Hadoop works best with Java 8). Install to a path without spaces if possible (e.g., `C:\Java\jdk8`), or use the short name `PROGRA~1` later.
* **Hadoop Binary:** Download `hadoop-x.y.z.tar.gz` from the Apache website.
* **Winutils:** Download the `winutils.exe` and `hadoop.dll` specific to your Hadoop version from a GitHub repository (e.g., `cdarlint/winutils` or `steveloughran/winutils`).

### 2. Setup Environment
1. **Extract Hadoop:** Unzip to a simple path (e.g., `C:\hadoop`).
2. **Install Winutils:** Copy `winutils.exe` and `hadoop.dll` into `C:\hadoop\bin`.
3. **Set Environment Variables:**
* `HADOOP_HOME` -> `C:\hadoop`
* `JAVA_HOME` -> `C:\Java\jdk8`
* **Path:** Add `%HADOOP_HOME%\bin` and `%HADOOP_HOME%\sbin`.
